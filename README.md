# ✨ Aura AI - Multi-Model Workspace

A premium desktop application built with Electron, designed to streamline your AI workflow. Whether you're an E-commerce professional, a language learner, or a power AI user, Aura AI provides a unified interface for all major LLMs.

![Aura AI Banner](https://raw.githubusercontent.com/ksrimalai01-sudo/Aura-Ai-Windows/main/assets/banner_placeholder.png)

## 🚀 Key Features

- **Multi-Panel Workspace**: Run Gemini, ChatGPT, Claude, and DeepSeek side-by-side.
- **Pro Mode (OpenRouter)**: Connect to hundreds of models via one API key with a sleek, native chat interface.
- **Smart Hub**: Send prompts to one AI or broadcast to all visible panels instantly.
- **Prompt Library**: Access professional-grade templates for E-commerce (Descriptions, Ads) and Japanese Learning (Grammar, Kanji).
- **Aura Dashboard**: A beautiful starting point to manage your workspace.
- **Cross-Platform**: Designed for Windows, fully compatible with macOS and Linux.
- **Cloud Sync**: Firebase integration for synchronizing settings and custom prompts across devices.
- **Power Features**:
    - ⚔️ **AI Debate**: Watch two AIs debate a topic to see different perspectives.
    - 🔗 **Workflow Chain**: Multi-step AI processing for complex tasks.
    - 📸 **Screenshot & Analyze**: Capture any part of your screen and send it directly to your preferred AI.

## 🛠 Tech Stack

- **Core**: Electron.js, HTML5, Vanilla CSS3 (Glassmorphism), JavaScript (ES6+).
- **Backend**: Firebase (Auth, Firestore).
- **AI Integrations**: Webview-based (Official sites) + OpenRouter API (Pro Mode).

## 💻 Installation

### For Windows (Setup)
1. Download the latest `.exe` from the [Releases](https://github.com/ksrimalai01-sudo/Aura-Ai-Windows/releases) page.
2. Run the installer and follow instructions.

### For Developers (Windows / Mac / Linux)
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Clone the repository:
   ```bash
   git clone https://github.com/ksrimalai01-sudo/Aura-Ai-Windows.git
   ```
3. Navigate to the folder:
   ```bash
   cd Aura-Ai-Windows
   ```
4. Install dependencies:
   ```bash
   npm install
   ```
5. Start the application:
   ```bash
   npm start
   ```

## 🍎 macOS Special Note
If you are running on Mac and encounter a "Developer cannot be verified" error:
- Right-click the app (or `Electron`) in Finder and select **Open**.
- Click **Open** again in the dialog box.
